## Final Project

### Acute Toxicity Prediction

급성 경구 독성 및 급성 흡입 독성 데이터를 활용하여 머신러닝 예측 모델을 구축하고 성능을 비교 평가함.

수행 내용

- 데이터 전처리 및 정제
- RDKit Descriptor 생성
- 머신러닝 모델 학습
- 모델 성능 비교 및 분석
- 최적 모델 선정
- 결과 시각화
