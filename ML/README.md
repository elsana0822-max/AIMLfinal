# AI Machine Learning

AI머신러닝 수업 실습 및 학습 과정을 기록하기 위한 목적의 파일 저장소

## 학습 내용

- 데이터 전처리 및 품질 검증
- 분자 데이터 처리 및 RDKit 활용
- Molecular Descriptor 생성
- Feature Selection
- Support Vector Machine (SVM)
- Artificial Neural Network (ANN)
- Tree-based Models
- Ensemble Learning
- Hyperparameter Optimization (Optuna)
- AutoML (FLAML)

## 사용 도구

- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- RDKit
